HTML => структура(разметка)
  не должно быть информации, которая влияет на отображение

  Чем плохая табличная верстка:
    - нельзя изменить без вмешательста в html, т.к уже задана структура (html влияет на представление)

  Хорошая верстка разделяет представление и структуру, с помощью css мы начинаем "раскрашивать"

CSS => оформление(представление)

tags- элементы на странице


block: div
inline: span

отличие
  блочный элемент по умолчанию занимает всю возможную ему ему ширину.
  линейный элемент занимает только необходимое ему место (столько, сколько занимает его содержимое), как пример буква
  линейным элементам нельзя назначить ширину и высоту

      наглядный пример
      <div>A</div> /* block-element div */
      <span>B</span> /* inline-element span */

с помощью display мы можем назначать линейным элементам свойства блочного

стили по умолчанию
  это стиль, который назначает браузер. Для того что б этого избежать используют reset css.
    * означает применить ко всему
недостатки
  например, когда нужно поддерживать блог(WYSIWYG) или что-то еще, где используются дефолтные стили, восстановить стили по умолчанию сложно

10px - 10 точек монитора
10% - от чего будем считать проценты (обычно считаются от 1-го родителя)
10em - высота строки (можно сбрасывать размер шрифта font-size 1 em) http://en.wikipedia.org/wiki/Em_(typography)
10pc - кегельная шпация http://en.wikipedia.org/wiki/Pica_(typography)

Full list:
em: the 'font-size' of the relevant font
ex: the 'x-height' of the relevant font

in: inches — 1in is equal to 2.54cm.
cm: centimeters
mm: millimeters
pt: points — the points used by CSS are equal to 1/72nd of 1in.
pc: picas — 1pc is equal to 12pt.
px: pixel units — 1px is equal to 0.75pt.

http://www.w3.org/TR/CSS2/syndata.html#length-units

Почему используем class, а не id
- id может быть только 1-н на старницу
- большой вес(чем из большего количества классов состоит селектор, тем больший вес он будет иметь)

Пример:

.block .heading {
  color: red;
}
.heading{
  color: white;
}

В результате, применится правило color: red; из-за веса селектора.
